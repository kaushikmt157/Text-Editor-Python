How to run:
install dependency tkinter
in terminal write pip install tkinter


Functionality:
File:  New, Open, Save As..., Close, Exit
Edit:  Undo, Cut, Copym Paste, Delete, Select All
Format:  Word Wrap
Tabs:  Right Click, New Tab Option, Center Click, Close Tab, Drag, Rearrange Tab

Keyboard Shortcuts:
Ctrl-s: Save
Ctrl-o: Open
Ctrl-w: Close tab
Ctrl-n: New tab
Ctrl-tab: Next tab
Ctrl-shift-tab: Previous tab
Ctrl-u: Undo
Ctrl-y: Redo

Description:
This is a GUI Text Editor made in Python using Tkinter module. It support many of the notepad features. The main aim of the program is to learn the concept of Tkinter.
